# 🕌 لغة البيان - Al-Bayan

أول لغة برمجة عربية أصلية تستمد قوتها من لغة القرآن.

## ✨ لماذا البيان؟
- إدارة الذاكرة بالضمائر
- التوازي بالأوزان (فَعَّلَ)
- غير المتزامن (فَاعَلَ)

## 🚀 مثال
```arabic
بَياناتِي = قَرَأَ ("ملف.txt")
النَتائِجُ = حَسَّبَ (بَياناتِي)
بَعَثَ (النَتائِجُ، "خادم_التقارير")
# 🕌 لغة البيان

cd al-bayan-compiler
echo "اِحتَسَبَ" > برنامجي.بيان
cargo run --bin bayan -- شغّل برنامجي.بيان
