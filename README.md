# 🕌 لغة البيان

اول لغة برمجة عربية اصلية. الكود قرآن.

## ابدأ في دقيقة

git clone https://github.com/Sham4753/al-bayan-compiler.git
cd al-bayan-compiler
echo "اِحتَسَبَ" > برنامجي.بيان
cargo run --bin bayan -- شغّل برنامجي.بيان

او جرب: http://178.105.83.75/دفتر_البيان.html
