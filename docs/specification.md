# وثيقة لغة البيان v0.5.0

## الهوية
- الاسم: لغة البيان (Al-Bayan)
- الإصدار: 0.5.0
- الشعار: الكود قرآن
- المخترع: محمد مجد الخطيب - حران العواميد
- الترخيص: MIT

## الفلسفة
البيان لغة برمجة عربية أصلية. كل كلمة جذر عربي، كل وزن نمط برمجي.

## الجذور
أصل الكلمة: ح-س-ب (الحساب)، ق-ر-أ (القراءة)، ح-ف-ظ (الحماية).

## الأوزان
- فعل: تنفيذ متزامن (حسب)
- فعل: معالجة متوازية (حسب)
- فاعل: غير متزامن (حاسب)
- استفعل: استدعاء خارجي (استقرأ)
- افتعل: مراقبة ذاتية (احتسب)
- انفعل: استماع (انبعث)

## الإعراب
- مرفوع (ضمة): فاعل
- منصوب (فتحة): مفعول به
- مجرور (كسرة): مدخل

## الجملة الفعلية
فعل + فاعل + مفعول به + جار ومجرور

## أدوات البرمجة
- متغيرات: اجعل س = قيمة
- دوال: دالة اسم(معامل) = جسم
- شرط: إذا شرط ف نتيجة
- تكرار: كرر فعل عدد مرات
- استثناء: حاول مهمة وإلا ف بديل

## مسار الترجمة
كود البيان ← محلل صرفي ← منسق ← مترجم C ← GCC ← Machine Code

## المكتبة القياسية
- اقرأ (ملف): قراءة
- اكتتب (ملف، محتوى): كتابة
- احسب (بيانات): معالجة
- شفر (بيانات): تشفير
- افحص (نظام): فحص
- ارسل (بيانات، عنوان): إرسال
- استقبل (منفذ): مستمع
