# 🕌 إضافة لغة البيان لـ VS Code

✨ **الكود قرآن** ✨

## الميزات

- 🎨 **تلوين ذكي**: الأفعال البليغة باللون الذهبي
- 💡 **إكمال تلقائي**: اقتراحات ذكية للجذور والأوزان
- 🏆 **تقرير البلاغة**: Ctrl+Shift+B
- 👤 **وضع المعلم**: تلميحات للمبتدئين
- ❌ **تدقيق نحوي**: كشف الأخطاء الإعرابية فورياً

## التثبيت

1. افتح VS Code
2. اذهب إلى الإضافات (Ctrl+Shift+X)
3. ابحث عن "لغة البيان"
4. اضغط تثبيت

## الاستخدام

- افتح ملفاً بامتداد `.bayan` أو `.بيان`
- ابدأ بكتابة `بسم` ثم اختر `بِسْمِ اللَّهِ`
- استمتع بالبرمجة بالعربية!
